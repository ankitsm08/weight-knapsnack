# Weight Knapsnack

A simple web app that helps you calculate the best bottle combination to hit your target weight. With features such as weight overshooting, and bottle penalty.

## Tech Stack

- **Backend**: Python 3.12.9 (Flask)
- **Frontend**: HTML, CSS, and JS (through Flask)

## Future Plans

- Rewrite the backend in FastAPI
- Rewrite the frontend in React

## Setup

1. Clone the repository, and navigate to the `weight-knapsnack` directory: 
    ```bash
    git clone https://github.com/ankitsm08/weight-knapsnack.git
    cd weight-knapsnack
    ```
2. Install `Python 3.12.9` and install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3. Run the Flask server by executing:
    ```bash
    python app.py
    ```
4. Access the web app through your browser at `http://localhost:5000`
5. Calculate the best bottle combination for your target weight using the web app interface.
